The CSV time_dynamics.csv file contains the time dynamics of opinions in the simulation.
Columns:
1. Time Step: The time step in the simulation.
2. Opinion 0 Count: Number of agents holding opinion 0 at that time step.
3. Opinion 1 Count: Number of agents holding opinion 1 at that time step.

Simulation Parameters:
Number of Agents: 50
Time Steps: 100
Proportion of Interactions per Time Step: 0.5

Generated by opinion dynamics simulation script.
